Opdracht 3: Er zijn verschillende selectors, bijvoorbeeld;
- type selectors
- descendant selectors
- class selectors

Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.


Class selectors: met class kan je aangeven hoe je het gestyled wilt hebben. Dus je verwijst het naar de class en daarmee dus de bijbehorende style sheet. Voorbeeld:.vet {property: waarde;}
Descendant selectors: deze stelt je in staat om je te beperken tot elementen die afstammeling (descendant) zijn van een ander element. Voorbeeld: div p { background-color: yellow;}
Type selectors: selecteert alle elementen die tot de genoemde titel behoren. Voorbeeld: span { background-color: skyblue;}
