Opdracht 2: CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

• De selector: Dit onderdeel duidt aan waar de opgemaakte tekst of stijlblok zich bevindt.
• De property identificeert in menselijke taal welk eigenschap het heeft.
• De value omschrijft welk waarde hoort bij de eigenschap.
